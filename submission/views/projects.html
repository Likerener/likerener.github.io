Project 3: Candy Land

  • The primary goal was to create an engaging and dynamic board game called "Candyland" implemented by using object-oriented programming principles in C++. Key elements included character selection, tile navigation with color-coded cards, stamina management, candy stores, and various gameboard events.
  • The game utilized arrays and vectors to store characters, candies, and board tiles, while file I/O was used to load game data dynamically. Random number generation played a crucial role in determining events such as hidden treasures and calamities. The objective was to make each game session unique by providing diverse characters, a variety of candies with different effects, and dynamic gameboard interactions, ensuring an engaging and unpredictable experience for players.
